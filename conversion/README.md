# Kgopolo Conversion Assessment

Service that converts the following :
* Kelvin to celsius
* Pounds to kilograms
* Miles to kilometers

## Installation


```bash
mvn clean install
```

## APIs

```Java SpringBoot
/api/celsius/{kelvin} # returns 'celsius value'
/api/kilograms/{pounds} # returns 'kilograms value'
/api/kilometers/{miles} # returns 'kilometers'
```
